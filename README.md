# AI Placement Assistant

AI-powered placement preparation platform that helps students improve resumes, analyze ATS scores, generate interview questions, and receive job recommendations.

## Features

- Resume Upload
- ATS Score Analysis
- ATS History Tracking
- Interview Question Generator
- Job Recommendations
- Resume Suggestions
- Admin Analytics Dashboard
- ATS Trend Charts

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS
- Recharts

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Deployment:
- Vercel
- Render
